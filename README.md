# FTX-ICO-Bot
Afraid of missing FTX ICO? It's ok, let's get ICO tokens as soon as possible.

## GettingStart
1. Edit some config settings in setting.json.example and rename it to setting.json.
2. Run program 1 min before the Announcement.

```
node main.js
```
